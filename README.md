# aps490-capstone-kite
### Dataset setup

Set the following environment variables after connecting Shared OneDrive folder to local computer:

**Windows (PowerShell)**
setx WINTERLAB_VIDEO_ROOT "C:\path\to\videos_renamed"
setx WINTERLAB_SPLIT_ROOT "C:\path\to\BaselineDataset"

**macOS (Terminal)**
export WINTERLAB_VIDEO_ROOT="/Users/username/path/to/videos_renamed"
export WINTERLAB_SPLIT_ROOT="/Users/username/path/to/BaselineDataset"

Restart VS Code after setting variables.
